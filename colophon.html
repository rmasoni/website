Colophon

This website is being built by hand on AstroNvim with good old HTML and CSS. Not using generators, CMSs and SaaS was a conscious decision inspired by Louie Mantia.

Type is set in iA Writer Duo.

Two Raycast extensions have been incredibly useful during development: UUID Generator to generate unique identifiers for the RSS feed, and Image Modification to convert images. Other tools I used can be found here.

Made on a MacBook Pro (13-inch, M1, 2020) connected to a Dell P2721Q (27-inch, 4k) display, while listening to communist lectures on Sony WH-1000XM5 noise-cancelling headphones.
